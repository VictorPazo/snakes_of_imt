# Snakes_Of_IMT

Repositório contendo o desenvolvimento de um projeto acadêmico para a conclusão do curso de Ciência da Computação que possui como objetivo desenvolver um aplicativo mobile que utiliza de Visão Computacional e possui como finalidade, classificar espécies de serpentes brasileiras a partir de imagens, fornecendo informações rápidas e educativas para a população do Brasil, especialmente em áreas rurais, auxiliando na identificação correta da espécie e nas ações iniciais a serem tomadas em casos de acidentes ofídicos

Funcionalidades:
*Identificação por imagem:* Classificação de espécies de serpentes brasileiras via captura de câmera ou upload da galeria do dispositivo.

**Tecnologias Utilizadas:**

Frontend
**Framework:** Flutter
**Linguagens:** Dart

Backend
**Linguagem:**Python
**Banco de dados:** AWS - Amazon Web Services
**Modelos de treinamento:** YOLO

- English Version:

Repository containing the development of an academic project for the completion of a Computer Science degree, which aims to develop a mobile application that uses Computer Vision. Its purpose is to classify Brazilian snake species from images, providing quick and educational information for the population of Brazil, especially in rural areas, assisting in the correct identification of the species and the initial actions to be taken in the event of snakebites.

Features:

- Image Identification: Classification of Brazilian snake species via camera capture or upload from the device's gallery.

Technologies Used:

Frontend

- Framework: Flutter
- Language: Dart

Backend

- Language: Python
- Database: AWS - Amazon Web Services
- Training Models: YOLO
